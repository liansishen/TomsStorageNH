# Tom's Simple Storage 1.7.10 Port

[简体中文](README_zh.md)

## Overview

Tom's Simple Storage 1.7.10 Port is a backport project that brings the modern Tom's Simple Storage mod experience to Minecraft 1.7.10.

This mod provides a lightweight centralized storage network, allowing players to access items from connected containers through terminals and enjoy an experience close to newer versions of Tom's Simple Storage in a 1.7.10 environment.

## Required Dependencies

- NotEnoughItems

## Features

- Use storage connectors to connect adjacent containers and build a centralized storage network.
- Use storage bridge blocks to extend the container connection range.
- Use storage terminals to view, insert, and extract items from the network in one place.
- Use crafting terminals to craft items while accessing the storage network.
- Use wireless terminals to remotely access a bound terminal within a limited range.
- Supports NEI integration for recipe viewing, recipe transfer, and auto-crafting.
- Optional compatibility with NotEnoughCharacters for improved Chinese and pinyin search.
